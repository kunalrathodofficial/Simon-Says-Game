Simon Says Game

Rules to Play

  • Game has four blocks labeled 1, 2, 3, and 4.
  • Each level shows a sequence of blinking blocks.
  • Start with one block in Level 1, adds one more each level.
  • Repeat sequence by clicking blocks in order.
  • Get it right, another block added to sequence.
  • Keep copying until you make a mistake.
  • Goal: Reach highest level by remembering sequence.
